Guess the word-of-the-day using ten guesses or less. Submit each guess by typing 4 letters, followed by the Enter key. (A guess is not required to be a valid word.) After each guess, a response is provided to the guess in the four circles to the right: A red circle indicates there is a correct letter in a correct position. A white circle indicates there is a correct letter, but not in the correct position.

If you'd like to play using a different word than the word-of-the-day, press the "Play random word" button.
